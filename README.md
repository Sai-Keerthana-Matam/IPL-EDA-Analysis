# 🏏 IPL Exploratory Data Analysis Dashboard

## 📌 Project Overview

This project performs Exploratory Data Analysis (EDA) on IPL match and ball-by-ball datasets to uncover insights about team performance, player achievements, venue trends, and toss impact.

The project also includes an interactive Streamlit dashboard for visualizing IPL statistics.

---

## 🎯 Objectives

* Analyze the most successful IPL teams
* Identify top-performing batsmen and bowlers
* Study Player of the Match trends
* Analyze toss decision patterns
* Explore venue statistics
* Build an interactive dashboard

---

## 📊 Key Insights

* Mumbai Indians are the most successful IPL franchise.
* Chennai Super Kings rank second in total wins.
* Winning the toss can influence match outcomes.
* Certain venues host significantly more matches.
* A small group of players dominate batting and bowling statistics.
